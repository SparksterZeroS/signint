# signint

*A set of utility functions for exchanging the signness of integers.*

## Usage

To use signint, simply include it in your project using the following techniques...

For browsers (iife):

```js
// W.I.P
```

For ESModules (mjs):

```js
import * as SignInt from "signint";
```

For CommonJS (cjs):

```js
const SignInt = require("signint");
```
